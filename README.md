## About
SASA HACK TOOL BOX is a **Python-based Termux tool installer**, inspired by DarkFly-style menu tools.  
It comes with multiple categories of tools like **OSINT, Network, Web, IP Scan, and more**.  
This tool is for **educational and ethical purposes only**.

---

## Features

- ✅ Colorful hacker menu UI  
- ✅ Python2 + Python3 support  
- ✅ Tool categories: OSINT, Network, Web, IP Scan, etc. (20+ tools)  
- ✅ ASCII hacker banner (ZORRO DARK TOOLS)  
- ✅ One-click install script  
- ✅ Easy to run in Termux
